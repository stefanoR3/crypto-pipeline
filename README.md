# Crypto Distributed Pipeline (Work in Progress)

This project is a distributed data infrastructure designed to ingest, process, and store real-time cryptocurrency market data. It serves as both a practical Data Engineering application and a study of distributed database systems.

## 🚧 Status: Work in Progress
Currently implementing the **Data Ingestion** layer.

## 🛠️ Tech Stack (Planned):
- **Language:** Python 3.x
- **Database:** PostgreSQL (Distributed architecture with Sharding/Replication)
- **Containerization:** Docker & Docker Compose
- **Data Source:** CoinGecko API

## 📈 Objectives:
- [ ] Build an automated ETL (Extract, Transform, Load) pipeline for crypto prices.
- [ ] Implement a distributed database system for scalability and fault tolerance.
- [ ] Set up process monitoring and logging for data reliability.
